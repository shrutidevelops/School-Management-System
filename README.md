# School Management System

## 📖 Description
The School Management System is a full-stack web application developed using PHP and MySQL. It is designed to manage student records, academic data, and administrative tasks efficiently in a structured and user-friendly way.

## 🚀 Features

- 🔐 **Login Authentication System**
- 🗂️ **Grade Management (CRUD Operations)**
  - Add new records
  - Update existing data
  - Delete records
  - Search records
- 🔎 **Search Functionality**
  - All records search
  - Particular search
  - Column-based search
- 🗄️ **Database Connectivity using MySQL**
- 🎨 **Clean UI using HTML & CSS**

---

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP
